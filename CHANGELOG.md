# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-05-26

### Added

- Initial public release preparation for NB Capture Suite.
- Windows capture workflow documentation.
- Windows PyInstaller build script.
- GitHub issue and pull request templates.
- CI and release workflow drafts.

### Known Limitations

- Windows is the only supported platform.
- macOS is not supported yet.
- Windows binaries are not code-signed unless the release owner provides a signing certificate.
- Binary distribution should still verify the PyQt/Qt licensing path.
